# 📊 Earnings Weekly Tracker

> Analyse quantitative hebdomadaire des **TOP 10 Earnings Plays** — Univers US, Europe & Grandes Capitalisations Internationales.

---

## 🎯 Objectif

Identifier chaque semaine les **10 sociétés cotées présentant le meilleur potentiel rendement/risque** avant publication de leurs résultats financiers.

## 📁 Structure

```
earnings-weekly-tracker/
├── README.md
├── weeks/

│   ├── 2026-W21_18-22-mai.md          ← Rapport W21 ✅ NEW

```

## 📐 Critères d'analyse

| Critère | Détail |
|---|---|
| Date résultats | Confirmée ou fortement probable dans les 7 jours |
| Consensus analystes | EPS attendu, CA attendu, révisions récentes |
| Historique surprises | Beat/Miss sur les 4 derniers trimestres |
| Momentum boursier | Perf. 1M / 3M / 6M + tendance technique |
| Valorisation | PER, PEG, EV/EBITDA |
| Solidité financière | Dette, marge nette, free cash flow |
| Sentiment marché | Upgrades/downgrades, insider buying, flux institutionnels |
| Risques | Volatilité, contentieux, guidance fragile |

## ⚖️ Score /100

Chaque société reçoit un **score global /100** basé sur :
- Qualité fondamentale
- Probabilité de beat earnings
- Réaction potentielle positive du marché
- Niveau de risque maîtrisé
